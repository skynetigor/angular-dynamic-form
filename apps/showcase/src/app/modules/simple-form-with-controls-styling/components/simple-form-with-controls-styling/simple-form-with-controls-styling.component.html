DynamicForm supports styling of several controls and styling of particular control.
Here is example how styling of controls works. All you need to do is to specify styles through ::ng-deep.
If you want to specify styles for all controls you need to do it in styles for elemnts that has '@skynet-ng/dynamic-form' attribute.
If you want to style particular control you can do it by styling '.#controlName#' class, where '#controlName#' is name that you create control with in formModel.
